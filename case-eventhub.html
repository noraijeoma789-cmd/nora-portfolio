EventHub

Event management portal with public event listings, attendee registration, and an organizer dashboard.

The Problem

An events company was managing event registrations through manual WhatsApp collection and Google Forms with no central system. Organisers had no live view of attendee numbers and feedback was collected separately with no connection to event records.

The Solution

I built a Softr portal backed by Airtable with three layers: public event listings, an attendee registration flow that writes to Airtable in real time, and an organizer dashboard showing registrations per event. An n8n workflow triggers a Gemini AI summary of post-event feedback and appends it to the event record automatically.
